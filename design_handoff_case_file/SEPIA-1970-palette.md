# Промпт для Claude Code — палитра «Сепия 1970-х» (финал)

Вставь весь файл в Claude Code как задачу.

---

## Контекст

Проект — «Где ложь? Симулятор детектива» (React + TypeScript + Tailwind + Framer Motion, Vite).
Экран = метафора «папка с делом на столе детектива»: бумага, бланки, чернильные штампы,
манильские папки. Аудитория 30–60, приоритет — читаемость при дневном свете.

**Задача:** заменить текущую тёплую «манильную» палитру на палитру **«Сепия 1970-х»** —
поджаренный архивный стол, состаренная бумага, жжёно-терракотовый акцент вместо бирюзы.
Вёрстку и структуру токенов НЕ трогать — менять ТОЛЬКО значения цветов во всех местах, где
они заданы (tailwind.config.js, CSS-переменные темы, инлайновые цвета, если есть).

> ВАЖНО: прошлая попытка почти ничего не изменила, потому что значения были слишком близки к
> маниле. Ниже — таблица «было → стало». Замени КАЖДОЕ старое значение на новое по всему репозиторию
> (grep по hex старых значений). Сдвиг намеренно заметный: стол глубже-янтарный, бумага явно
> состаренная (слоновая кость), акцент из бирюзы превращается в жжёную сиену.

## Жёсткие правила

- Шапка-баннер бланка («ЗАЯВЛЕНИЕ КЛИЕНТА · Форма СТ-1») и бейджи ФОТО/ДОК: фон `--folder-edge`
  (`#876520`), текст `--paper` (`#F6ECD6`).
- Кнопки вердикта: «ОТКЛОНИТЬ» = заливка `danger`, текст белый; «ОДОБРИТЬ» = заливка `success`,
  текст белый.
- `gold` — ТОЛЬКО заливка/рамка/иконка (бейдж «СРОЧНО», левая рамка «дела дня»). Как цвет текста на
  светлом фоне НЕ использовать — для текста брать `text-light`.
- `accent` (`#95431F`) — ссылки «Открыть досье →», XP-бар, числа баланса/точности, активная галочка.
  Как кнопка — белый текст.
