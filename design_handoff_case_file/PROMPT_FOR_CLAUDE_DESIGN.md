# Промпт для Claude Design — дополнить макет «Где ложь? Симулятор детектива»

> Скопируй всё, что ниже разделителя, и вставь в Claude Design вместе с файлом
> `ClaimGame.dc.html` (главный референс) из этого же пакета.

---

Ты дорабатываешь существующий hi-fi макет браузерной игры **«Где ложь? Симулятор детектива»**
(страховой следователь, платформа Yandex Games). Главный референс — приложенный
`ClaimGame.dc.html` (формат Design Component: разметка между `<x-dc>…</x-dc>`,
логика в `<script data-dc-script>`). **Не меняй уже существующие экраны** — они
утверждены. Нужно **добавить недостающие фичи**, которые уже есть в проде, но
которых нет в макете, выдержав ту же визуальную метафору и те же токены.

## Жёсткие ограничения (метафора неизменна)

«Папка-дело на столе следователя»: бумажные документы, официальные формы,
чернильные штампы. Аудитория 30–60 лет. Приоритет — читаемость, минимализм,
крупные зоны нажатия, спокойный интеллектуальный темп. **Запрещено:** неон,
мультяшность, игровые/sci-fi градиенты, эстетика мобильных казино, фанфары.
Любая новая механика должна выглядеть как **канцелярский/архивный артефакт**, а
не как «ачивка из мобильной игры».

## Токены (используй ровно их)

- Фон стола `#111827`; панели `#1F2937`; вложенные блоки `#111827`; границы `#374151`.
- Бумага: Манила `#F5F1E8`, Досье `#F3F4F6`. Текст на бумаге `#1F2937`.
- Текст на тёмном: основной `#E5E7EB`, вторичный `#9CA3AF` / `#6B7280`.
- Акцент `#3B82F6` (Манила) / `#2563EB` (Досье). Успех `#16A34A`. Опасность `#DC2626`.
- Золото дня `#D9A441`, тёмное золото `#3A2705`, текст-золото `#F0C46B`.
- Штамп `#B4231F` (Манила) / `#DC2626` (Досье).
- Две темы папки `folderLook: manila | dossier` уже заданы — новые блоки тоже
  должны переключаться через те же CSS-переменные `--folder`, `--folder-edge`,
  `--paper`, `--accent`, `--stamp`.
- Шрифты: **IBM Plex Serif** (документы/заголовки/нарратив), **Inter** (UI-чром),
  **IBM Plex Mono** (архивные данные: чек, GPS, номера дел, цифры баланса/XP,
  тексты штампов). Радиусы: панели 12, документы/карточки 6–10, чипы 4–5, кнопки 9.

## Что нужно добавить

### 1. Карьера: ранги и XP (левый сайдбар, низ)

В текущем макете внизу левого сайдбара уже есть карточка «Следователь · ур. 14 ·
XP-бар». Расширь её до полноценной системы рангов:

- Лестница из 6 рангов (i18n-подписи): **Стажёр → Мл. инспектор → Инспектор →
  Ст. инспектор → Ведущий следователь → Главный следователь**
  (пороги XP: 0 / 50 / 150 / 350 / 700 / 1200; бонус к награде +0/+2/+5/+8/+12/+18 %).
- Карточка: строка «Следователь» (Inter) + текущий ранг (акцентом справа), бар
  прогресса до следующего ранга (`--accent`), подпись моно «{XP в ранге} / {до
  следующего} XP до повышения». На максимальном ранге — просто «{XP} XP».
- XP — **перманентная** прогрессия (не тратится), валюта `balance` — отдельная.

### 2. Повышение в звании (на экране результата)

После закрытия дела, если XP перешагнул порог ранга — показать **штамп-плашку
«ПОВЫШЕНИЕ»** на листе результата (под разбором награды): чернильный штамп
золотом `#D9A441`, rotate ≈ −3°, анимация впрыгивания (как у штампа-резолюции),
второй строкой — название нового ранга (serif). Тихо, по-канцелярски, без салюта.

### 3. Достижения (3 точки появления)

Одноразовые ачивки, каждая даёт разовый бонус XP + ₽. Текущий набор (иконка +
заголовок + описание): 🕵️ первое раскрытое мошенничество · 📁 10 дел · 🎯
идеальные доказательства на сложном деле · 🔥 серия из 5 дней · ✋ 10 дел без
ложных отметок. Нужны:

- **Кнопка-архив** в правом сайдбаре (рядом с аналитикой): «Достижения · N / M»
  (моно-счётчик), стиль — как прочие карточки правого сайдбара.
- **Модалка-архив** (как модалка улики: тёмная шапка `--folder-edge` + ✕ +
  бумажное тело): список карточек. Открытая ачивка — рамка/фон золотом
  `#D9A441` + иконка; закрытая — `🔒`, opacity .6, grayscale. Подпись моно
  «+{XP} опыта · +₽{бонус}».
- **Плашка-уведомление на экране результата**: для только что открытых ачивок —
  узкая золотая карточка «Достижение получено» + иконка + название + бонус.

### 4. Подсказки в деле (Investigation Layer)

Под сеткой улик (перед панелью вердикта) — **панель «ПОДСКАЗКИ»** (тёмная,
`#1F2937`, рамка `#374151`). Две крупные кнопки-зоны:

- **«Заметка инспектора»** — платно из баланса (20 % от суммы иска); справа
  цена моно «₽{сумма}»; рамка акцентом; недоступна (dim), если не хватает денег.
- **«Опрос свидетелей»** — бесплатно, по просмотру **rewarded-видео Yandex**;
  справа «▶ Реклама»; рамка золотом.
- Обе **раскрывают истинный статус** одной ещё не раскрытой улики. После раскрытия
  на карточке улики и в модалке улики появляется **бейдж-вердикт**: «⚠
  Противоречие» (на `--stamp`-фоне) или «✓ Без противоречий» (на `success`-фоне),
  моно, мелкий. Когда раскрыты все улики — панель заменяется строкой «Все улики
  раскрыты».
- Нарисуй состояния: обе кнопки активны / «Заметка» недоступна (нет денег) / все
  раскрыты. И вид карточки улики и модалки **с бейджем раскрытия**.

### 5. Серия дней (streak)

Бонус за последовательные дни с закрытым делом: +5 %/день к награде, потолок
+50 %. В правом сайдбаре — компактная карточка-строка: 🔥 «Серия: {N} дн.»
(золотом). Показывать только когда серия активна (> 1). Спокойный значок, без
анимации пламени.

### 6. Расширённый разбор награды (экран результата)

Текущий разбор «Верный вердикт +50 % / Чек +25 % / GPS +25 %» дополни строками,
когда они применимы (тот же стиль строк с кружком-маркером ✓/—):

- **Бонус (+{rank%+streak%})** — золотой маркер, значение «+{%}». Применяется
  только к положительной базе (вердикт + доказательства), не к штрафу.
- **Штраф** — за ложно отмеченные улики, −50 ₽ за каждую. Маркер опасности «!»,
  значение красным «− {сумма}».
- **Дело дня ×5** — если применился множитель.
- Строка **«Опыт +{XP}»** (моно, акцентом) — отдельной плашкой под плитками
  «Гонорар» / «Баланс компании».
- Итоговый «Гонорар» может быть **отрицательным** (красным), если штраф перевесил.

### 7. Банкротство → восстановление средств

Если баланс ≤ 0 — **полноэкранный гейт** (бэкдроп `rgba(8,11,17,.8)`, бумажный
лист по центру): заголовок «Средства закончились», поясняющий текст, и крупная
кнопка с акцентом «▶ Восстановить средства (₽2000)», открывающая **rewarded-видео
Yandex**. Прогресс заблокирован, пока не восстановят.

### 8. Пауза по рекламе (ad pause guard)

Любое открытие/закрытие рекламы (fullscreen или rewarded) ставит игру на паузу.
Нарисуй **оверлей-паузу**: затемнение `rgba(0,0,0,.9)`, по центру спокойный знак
паузы «⏸». Без текста, без анимации.

### 9. Список дел (расширение)

В проде ~15+ дел (не 2). В левом сайдбаре «ДЕЛА В РАБОТЕ» покажи паттерн для
длинного списка: активное дело (рамка `--accent` + точка), пройденное (галочка
`✓` `success`), доступное (обычное), заблокированное (opacity .55). Список
скроллится. Тулзы вроде поиска/фильтра не нужны — только состояния карточек.

## Формат сдачи

- Тот же формат прототипа (`*.dc.html`), та же логика состояний и те же анимации
  (`stampIn`, `modalIn`, `riseIn`, прогресс-бар `width .9s`).
- Каждую фичу покажи во **всех релевантных состояниях** (см. выше) и в **обеих
  темах** (Манила / Досье), десктоп + мобайл.
- Все строки — выноси в словарь (i18n), языки: `ru, en, tr, ar, kk`; основной
  язык прототипа — русский. Учитывай RTL для `ar`.
- Никаких реальных ассетов — фото/карты/полароиды остаются плейсхолдерами
  (полосатый `repeating-linear-gradient` + моно-подпись).
