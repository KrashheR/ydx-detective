# Хэндофф для Claude Code — переверстать ДЕСКТОП экрана стола на раскладку «Доска · сетка» (Концепт C)

Стек проекта: **React + TypeScript + Tailwind + Framer Motion + Vite**.
Меняем **только центральную колонку десктопа** на экране выбора дела. Левый сайдбар («Где ложь?», язык, «ДЕЛА В РАБОТЕ», профиль) и правую «Аналитику» НЕ трогаем. Мобайл НЕ трогаем (там остаётся текущая подача).

Эталон визуала: блок **«C · ДОСКА · СЕТКА»** в файле `Claim Detective - Стол · раскладки папок.dc.html` (открыть в превью). Этот документ — текстовая спецификация того же.

---

## 1. Что меняем по сути

Сейчас центр — одна вертикальная колонка папок (по одному делу на экран, бесконечный скролл на 38 дел).
Становится:
1. **Герой «Дело дня»** — крупная золотая папка во всю ширину центра, сверху.
2. **Сетка папок 2-в-ряд** (`grid-cols-2`) — все остальные дела компактными карточками-папками. Видно ~6 дел сразу, виден прогресс/блокировки.

Заголовок секции остаётся: `ВЫБЕРИТЕ ДЕЛО ДЛЯ РАССЛЕДОВАНИЯ` (caps, letter-spacing, muted).

## 2. Токены (сепия, как в текущем билде)

Цвета папок зависят от статуса дела `kind: 'daily' | 'active' | 'locked'`:

| Роль | daily | active (доступно) | locked |
|---|---|---|---|
| Обложка `bg` | `linear-gradient(135deg,#e6b052,#d6982f)` | `#d4b558` | `#d3ccbb` |
| Вкладка/край `edge` | `#b07d1f` | `#a98a33` | `#b3ab98` |
| Текст-заголовок `ink` | `#3a2705` | `#2e2207` | `#6b6350` |
| Вторичный текст `sub` | `#7a5410` | `#6e5a24` | `#8b8470` |

Прочее:
- Фон центра (десктоп-стол): `#e8e1d0`, рамка контейнера `#cdbf9f`, радиус `10px`.
- Стамп-красный (СРОЧНО): `#b4231f`, текст белый.
- Чип «ДОСТУПНО»: текст `#2e6b3f`, фон `rgba(46,107,63,.12)`, рамка `rgba(46,107,63,.3)`.
- Чип «ЗАКРЫТО» (locked): текст `#8b8470`, фон `rgba(0,0,0,.05)`, рамка `rgba(0,0,0,.10)`.
- Пипсы сложности (3 точки): активные `#7a5410` (locked: `#9b937f`), пустые `rgba(0,0,0,.12)`. `diff: 1|2|3` → сколько точек закрашено.

Шрифты (уже в проекте): заголовки папок — `Spectral` 700; номера/чипы/пипсы — `JetBrains Mono`; остальное — `Inter`.

## 3. Разметка / компоненты (React + Tailwind)

```
<DeskCenter>                     // центральная колонка
  <h2> ВЫБЕРИТЕ ДЕЛО ДЛЯ РАССЛЕДОВАНИЯ </h2>
  <DailyHeroFolder case={dailyCase} />        // full-width
  <div className="grid grid-cols-2 gap-[13px] mt-[14px]">
    {cases.map(c => <CaseFolderCard key={c.id} case={c} />)}
  </div>
</DeskCenter>
```

### 3.1 DailyHeroFolder (герой «Дело дня»)
- Контейнер: `relative overflow-hidden rounded-[9px] p-5 cursor-pointer`, фон `linear-gradient(135deg,#e6b052,#d6982f)`, рамка `1px solid #b07d1f`, тень `0 12px 28px rgba(40,28,10,.2)`.
- Лента «СРОЧНО»: абсолютная, top:15 right:-34, `rotate(34deg)`, фон `#b4231f`, белый, mono 800, letter-spacing `.16em`, padding `5px 44px`.
- Строка: `СРОЧНОЕ ДЕЛО ДНЯ · {таймер}` (mono, `#5c3f08`).
- Заголовок: `Spectral 700 24px`, `#3a2705`.
- Чипы: `Награда ×{reward}` (фон `#3a2705`, текст `#f0c46b`), `Сложность ×{diffMult}` (фон `rgba(58,39,5,.16)`, текст `#3a2705`), `{docs} документа` (текст `#5c3f08`).

### 3.2 CaseFolderCard (карточка-папка в сетке)
Структура «папка с вкладкой» — два элемента:
1. **Вкладка**: `w-24 h-[15px] rounded-t-[6px] ml-[14px]`, фон = `edge` статуса.
2. **Тело**: фон = `bg` статуса, рамка `1px solid {edge}`, `rounded-[2px_8px_8px_8px]`, `p-4`, тень `0 7px 16px rgba(40,28,10,.16)`, `overflow-hidden`.

Внутри тела:
- Верхняя строка (`flex justify-between items-start gap-2`):
  - слева: `{num}` (mono 9px, sub) + `{title}` (Spectral 700 16px, ink);
  - справа: **чип статуса** (`ДОСТУПНО` для active / `ЗАКРЫТО` для locked), см. токены чипов.
- Мета-строка (`Inter 11px`, sub): если есть клиент → `{client} · {sum}`; если locked → текст условия (`{lock}`, напр. «Откроется на ур. 2»); иначе → `{docs}`.
- Низ (`flex justify-between items-center`, верхняя граница `1px` — active `rgba(0,0,0,.15)`, locked `rgba(0,0,0,.10)`):
  - слева: 3 пипса сложности + подпись `{diffLabel}` (mono 9px);
  - справа: CTA — active: `Открыть →` (`#2e2207`); locked: `заблокировано` (`#9b937f`).
- Для locked всю карточку приглушить: `opacity:.72`.

### 3.3 Hover (Framer Motion или CSS)
- Карточка: `translateY(-5px)` + тень `0 18px 34px rgba(40,28,10,.32)`, transition `.16s`. Locked карточки тоже можно поднимать (или оставить статичными — на ваш вкус).
- Курсор `pointer` на доступных; на locked — `not-allowed` и клик показывает тост с условием разблокировки (как сейчас).

## 4. Модель данных дела (TS)

```ts
type CaseKind = 'daily' | 'active' | 'locked';
interface DeskCase {
  id: string;
  num: string;            // 'Дело 1'
  title: string;          // 'Затопленная квартира'
  kind: CaseKind;
  client?: string;        // 'Игорь Семёнов'
  sum?: string;           // '1 200 ₽'
  docs: string;           // '2 документа'
  diff: 1 | 2 | 3;        // закрашенных пипсов
  diffLabel: string;      // 'Лёгкое' | 'Среднее' | 'Сложное'
  reward?: number;        // только daily (×5)
  lock?: string;          // только locked: 'Откроется на ур. 2' | 'Закройте предыдущее дело'
}
```
`dailyCase` рендерится героем; `cases` (без daily) → сетка. Сортировка сетки: сначала `active`, затем `locked` по порядку.

Хелпер цвета:
```ts
const folderColors = (k: CaseKind) =>
  k === 'daily'  ? { bg:'linear-gradient(135deg,#e6b052,#d6982f)', edge:'#b07d1f', ink:'#3a2705', sub:'#7a5410' }
: k === 'active' ? { bg:'#d4b558', edge:'#a98a33', ink:'#2e2207', sub:'#6e5a24' }
:                  { bg:'#d3ccbb', edge:'#b3ab98', ink:'#6b6350', sub:'#8b8470' };
```

## 5. Сетка и адаптив
- Базово `grid-cols-2 gap-[13px]`. Центр узкий → допустимо `grid-cols-1`; центр широкий (≥ ~1100px) → можно `xl:grid-cols-3`.
- Контейнер центра: `bg-[#e8e1d0] border border-[#cdbf9f] rounded-[10px] p-[22px]`.
- Скролл — внутри центральной колонки (как сейчас), сетка прокручивается; левый/правый сайдбары фиксированы.

## 6. Доступность
- Каждая папка — `<button>`/`role="button"`, `aria-disabled` для locked, видимый `:focus-visible` контур.
- Контраст ink↔обложка проходит AA (ink тёмный, обложки светлые/золотые).
- Чипы и пипсы дублируются текстом (`diffLabel`, `ДОСТУПНО/ЗАКРЫТО`) — не только цветом.
