# Хэндофф: раскладка стола «Доска · сетка» (Концепт C)

Пакет для передачи в Claude Code, чтобы переверстать **десктоп** экрана выбора дела.

## Что внутри
- **DESK-concept-C-handoff.md** — основная спецификация (промпт для Claude Code): токены, компоненты, TS-модель, состояния, a11y, definition of done. Это главный файл — его содержимое можно целиком вставить в Claude Code как задание.
- **concept-C-эталон.png** — скриншот-эталон визуала. Приложите его в Claude Code, чтобы он видел картинку.
- **визуал-раскладки-папок.html** — автономный интерактивный визуал (открывается в любом браузере, работает офлайн). Содержит все три концепта; нужен блок «C · ДОСКА · СЕТКА».

## Как передать в Claude Code
1. Открой задачу в Claude Code в репозитории игры.
2. Вставь текст из `DESK-concept-C-handoff.md` как описание задачи.
3. Приложи `concept-C-эталон.png` (и при желании `визуал-раскладки-папок.html`) как референс.
4. Уточни: менять только десктоп-центр, сайдбары и мобайл не трогать.
