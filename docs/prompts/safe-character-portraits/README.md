# Безопасные портреты заявителей

В этой папке лежат самостоятельные промпты для 12 портретов, удалённых во время IP-аудита.
Каждый файл можно передать генератору отдельно. Сохраняй результат под именем из поля
`Output filename` в `public/people/`, затем добавляй в соответствующее дело:

```json
"personImage": "people/<output-filename>"
```

Требования ко всей серии:

- квадрат 1:1, рекомендуемый финальный размер 360×360 WebP;
- голова и плечи, строго анфас;
- сдержанная гуашь/акварель на тёплой архивной бумаге;
- тонкая двойная коричневая рамка внутри изображения;
- оригинальное лицо без сходства с реальными людьми или известными персонажами;
- без текста, логотипов, эмблем, документов, формы, костюмов и водяных знаков.

После добавления всех файлов проверь, что ни одно дело не осталось без `personImage`, и запусти
`npm run typecheck` и `npm test`.
