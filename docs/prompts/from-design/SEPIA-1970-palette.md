# Промпт для Claude Code — перевод игры на палитру «Сепия 1970-х»

Вставь весь этот файл в Claude Code как задачу.

---

## Контекст

Проект — «Где ложь? Симулятор детектива» (React + TypeScript + Tailwind + Framer Motion, Vite).
Экран — метафора «папка с делом, разложенная на столе детектива»: физическая бумага,
официальные бланки, чернильные штампы, манильские папки. Аудитория 30–60 лет, приоритет —
читаемость текста при дневном свете.

**Задача:** заменить текущую тёплую «манильную» палитру на палитру **«Сепия 1970-х»** —
поджаренная архивная бумага, премиальное ретро, терракотовый акцент. Метафору, вёрстку и
структуру токенов НЕ менять — только значения цветов. Все пары текст/фон проверены на WCAG AA+.

---

## 1. Фиксированный «хром стола» — `tailwind.config.js`

```js
// theme.extend.colors
colors: {
  bg:           '#E5D7BD', // фон стола (поджаренная бумага)
  surface:      '#F1E7D1', // сайдбары / панели
  'surface-2':  '#D7C6A5', // вложенные блоки внутри панелей
  border:       '#C5AF87', // границы панелей
  ink:          '#2A2117', // текст на белой бумаге документа
  'text-light': '#3D3119', // основной текст на хроме стола
  'text-muted': '#6B5733', // вторичный / приглушённый текст
  'text-dim':   '#6B5733', // = text-muted
  success:      '#5C7A33', // Approve (оливковая зелень под ретро)
  danger:       '#B23A2E', // Reject
  gold:         '#C98A2E', // подсветка ежедневного дела (только заливка/рамка, не текст)
}
```

## 2. Тема папки `manila` — CSS-переменные

```css
:root, [data-folder-theme="manila"] {
  --paper:        #F7EFDD; /* листы документов */
  --accent:       #884A28; /* основные действия / кнопки / ссылки (тёмная терракота) */
  --folder:       #CBAA6C; /* обложка папки */
  --folder-edge:  #8A6A30; /* тёмная вкладка папки + шапка-баннер бланка */
  --stamp:        #A8201C; /* чернильный штамп «противоречие» */
}
```

## 3. Прочее

```css
--scrollbar: #C9B089;
/* тултипы остаются тёмными, чтобы читаться поверх стола: */
--tooltip-bg: #2B2018;  --tooltip-border: var(--stamp);
/* хайрлайн ВНУТРИ бумаги (рамка блока «Сумма иска», разделители): */
--paper-line: #E4D6B6;
```

## 4. Жёсткие правила при правке

- Шапка-баннер бланка («ЗАЯВЛЕНИЕ КЛИЕНТА · Форма СТ-1») = фон `--folder-edge` (#8A6A30),
  текст `--paper` (#F7EFDD). Бейджи ФОТО/ДОК — так же.
- Кнопки вердикта: «ОТКЛОНИТЬ» = заливка `danger`, текст белый; «ОДОБРИТЬ» = заливка `success`,
  текст белый. (Оба ≥ 4.9:1.)
- `gold` использовать ТОЛЬКО как заливку/рамку/иконку (бейдж «СРОЧНО», левая рамка «дела дня»),
  НИКОГДА как цвет текста на светлом фоне — для текста брать `text-light`.
- `accent` (#884A28) — для ссылок «Открыть досье →», XP-бара, чисел баланса/точности, активной
  галочки. Как текст всегда на `surface`/`bg`. Как кнопка — белый текст (6.9:1, AAA).

## 5. Контраст (проверено)

| Пара | Контраст | Уровень |
|---|---|---|
| `ink` #2A2117 → `paper` #F7EFDD | 12.0:1 | AAA |
| `text-light` #3D3119 → `bg` #E5D7BD | 8.9:1 | AAA |
| `text-light` #3D3119 → `surface` #F1E7D1 | 8.6:1 | AAA |
| `text-muted` #6B5733 → `surface` #F1E7D1 | 4.9:1 | AA |
| `text-muted` #6B5733 → `bg` #E5D7BD | 4.9:1 | AA |
| `accent` #884A28 → `surface` #F1E7D1 | 4.6:1 | AA |
| белый → кнопка `accent` #884A28 | 6.9:1 | AAA |
| белый → кнопка `success` #5C7A33 | 4.9:1 | AA |
| белый → кнопка `danger` #B23A2E | 5.9:1 | AA |
| `paper` #F7EFDD → бейдж `gold` #C98A2E (текст `ink`) | 5.4:1 | AA |

> Изменения относительно прошлой версии сепии: `text-muted` затемнён #8A744A → **#6B5733**,
> `accent` затемнён #9C5A33 → **#884A28** — иначе мелкие подписи («ОТДЕЛ РАССЛЕДОВАНИЙ»,
> «ДЕЛА В РАБОТЕ») и ссылки не дотягивали до AA (были 3.2–4.4:1).
