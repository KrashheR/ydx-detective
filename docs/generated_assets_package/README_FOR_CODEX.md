# Готовые ассеты для подключения

Скопировать каталог `public` в корень проекта с сохранением структуры.

Затем применить привязки из `asset-bindings.json`:

- для `case-001` установить `personImage: "people/igor-semyonov.webp"`;
- для каждой улики найти объект по сочетанию `caseId + evidenceId` и установить
  `data.imageUrl` в значение `runtimePath` из манифеста;
- не переименовывать файлы: имена уже совпадают с ID дел и улик;
- не встраивать изображения как base64 и не переносить текст поверх изображений.

Ожидаемые размеры:

- `igor-semyonov.webp` — 359 × 359;
- все изображения улик — 1600 × 1000 (16:10).

После подключения проверить отсутствие 404 для шести `runtimePath` и открыть дела
1, 4, 14, 27, 37 и 49 кампании (внутренний ID последнего — `case-051`).
