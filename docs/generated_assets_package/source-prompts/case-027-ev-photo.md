# Дело 027 · Фотография затопленных стеллажей

Создай реалистичную документальную фотографию внутри сельскохозяйственного склада после ливня, горизонтальный кадр 16:10. Через открытые ворота виден серый вечерний свет. На переднем плане и в центре — металлические складские стеллажи с мешками семенного зерна; нижние ярусы намокли и частично находятся в тонком слое мутной воды. На стойках заметна неровная мокрая отметка уровня воды, выше неё мешки сухие и чистые. Пол покрыт отражениями, рядом лежит один слегка размокший картонный ярлык без читаемого текста. Кадр должен ясно показывать, что повреждены именно нижние партии, а не весь склад. Стиль: спокойная инспекторская фиксация, натуральные материалы — металл, мешковина, мокрый бетон, без людей и постановочности.

Не добавлять логотипы, маркировку с читаемым текстом, водяные знаки, дату, интерфейс камеры или фантастические эффекты.
