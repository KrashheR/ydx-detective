# Дело 004 · Фото следов с камеры инспектора

Создай реалистичную документальную фотографию для страхового расследования, горизонтальный кадр 16:10. Раннее утро после дождя на небольшой ферме: мокрая земля во дворе, отчётливая цепочка крупных коровьих следов ведёт от открытого загона к покосившемуся сеновалу, затем в сторону повреждённой теплицы. На заднем плане — край лёгкой тепличной кровли с выбитым стеклом и несколько разбросанных тюков сена; в воздухе влажная прохлада, без драматизации. Камера на уровне глаз инспектора, следы занимают передний план и образуют ясную ведущую линию. Цвета приглушённые, естественные: мокрая охра земли, серо-зелёная трава, белёсая плёнка теплицы. Стиль: подлинный рабочий снимок инспектора, слегка неровная композиция, реалистичная резкость и мягкий утренний свет.

Не добавлять людей, животных, логотипы, водяные знаки, подписи, дату, интерфейс камеры или читаемый текст.
