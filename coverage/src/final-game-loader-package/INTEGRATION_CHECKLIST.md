# Integration checklist

- [ ] Скопирована папка `src/components/GameLoader`.
- [ ] Импортируется `GameLoader.css` через сам компонент.
- [ ] В проекте установлен `framer-motion`.
- [ ] Loader монтируется выше основного игрового интерфейса.
- [ ] `visible` не зависит только от таймера.
- [ ] `sdkReady` учитывает офлайн/dev-фолбэк.
- [ ] `saveHydrated` выставляется после облачного сейва или LocalStorage.
- [ ] `casesValidated` выставляется после Zod-валидации.
- [ ] `assetsReady` ждёт только критические ассеты первого экрана.
- [ ] После 100% loader задерживается примерно на 350 мс.
- [ ] Во время loader основной UI не принимает клики.
- [ ] Текущий язык передаётся в `locale`.
- [ ] Для арабского проверена RTL-композиция.
- [ ] Проверены desktop, mobile portrait и low-height landscape.
- [ ] Loader не вызывает повторную загрузку ассетов при каждом рендере.
- [ ] Нет конфликта `z-index` с рекламным/паузным оверлеем.
