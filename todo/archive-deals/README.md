# Archive Deals Prompts

Черновые промпты для тематических архивов. Здесь не используются прямые названия франшиз, имена персонажей, цитаты, логотипы, гербы, узнаваемые реплики или точные копии ключевых сцен.

Общие правила для всех наборов:

- Держать ощущение "все понимают отсылку", но материал должен работать как самостоятельная оригинальная вселенная.
- Системная рамка игры не меняется: это архив страховых расследований, бумажные досье, официальные бланки, штампы, фото-улики, выписки, журналы, показания.
- Не использовать защищенные названия, имена, эмблемы, буквенные обозначения орденов, школ, кланов, планет, артефактов и команд.
- Не копировать знаковые композиции кадров, костюмы один в один, фирменные фразы или уникальные сюжетные повороты.

Общие правила для изображений:

- Стиль проекта: серьезный архив страхового расследования, физические материалы, спокойная документальная подача, высокая читаемость, без неона и без "фан-арта".
- Любые изображения для дел должны выглядеть как улики: фото с телефона, CCTV, сканы документов, служебные карточки, технические распечатки.
- Никаких логотипов, водяных знаков, читаемых вывесок, узнаваемых гербов, масок, эмблем, цитат или лиц известных персонажей.
- Финальный экспорт: `.webp`, оптимизированные файлы без заметных артефактов, под веб.

Файлы:

- `space-opera-pack.md` — космоопера с вайбом межзвездной саги в далекой далекой галактике.
- `magic-academy-pack.md` — магический архив с вайбом закрытой школы и бюрократии чудес.
- `sewer-vigilantes-pack.md` — городской архив с вайбом черепашьей команды, их руководителя крысы, любви к пицце и подземного клана.
