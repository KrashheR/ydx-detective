Что изменилось в v2
Изменения ThematicPacksModal относительно текущей реализации. Всё остальное — без изменений.
ИЗМЕНЕНИЯ СТРУКТУРЫ
▲ ЛЕВАЯ КОЛОНКА · PackList
Было: карточки-«папки» (~80px).
Стало: тонкие строки (52px) с цветным spine-bar (8px) слева.
Убрать: описание и «Открыть за рекламу» изнутри карточки.
Добавить кнопку «Все архивы» внизу левой панели.
▲ ПРАВАЯ КОЛОНКА · CaseList
Было: grid 2×N. Стало: flex-column, 1 дело = 1 строка (48px).
В строке: номер-кружок → название → статус-чип. Кнопка «Играть» только у открытого.
▲ ACTION ZONE
Убрать: блок «купить всё» из action-зоны — он теперь в левом сайдбаре. Оставить: 2 кнопки + ссылка «Восстановить».

НОВЫЙ ЭЛЕМЕНТ · spine-bar
8px полоска слева в строке пака. Цвет = цвет обложки архива. Только в PackList. Передаётся через pack.spineColor
МОБАЙЛ · АДАПТАЦИЯ
▲ НАВИГАЦИЯ ПО ПАКАМ
Было: вертикальный список.
Стало: горизонтальный скролл-ряд таб-чипов в хедере. Активный: rgba(246,236,214,.18) + border 1.5px.
▲ LAYOUT
2-колоночного грида нет — всё в одну колонку. Header (тёмный) → description → case list → sticky action bar.
▲ ACTION BAR
position: sticky; bottom: 0. 2 кнопки + сноска «навсегда». Кнопку «Все архивы» убрать — пространства нет.
БЕЗ ИЗМЕНЕНИЙ
Состояния строк дел (open / next / locked) — одинаковые на моб и десктоп.
Логика покупки и рекламной разблокировки — не меняется.
RTL: dir=«rtl» + spine-bar переезжает на правую сторону строки.
BREAKPOINT
Мобайл: width < 640px
Десктоп: 2-колоночный grid, левая 220px, правая flex:1.
